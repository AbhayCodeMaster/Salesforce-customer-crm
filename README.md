# Salesforce Customer CRM Project

## Overview
This project is a basic Customer Relationship Management (CRM) system built using Salesforce.

## Features
- Custom Object: Customer
- Fields:
  - Customer Name
  - Email
  - Phone
  - Status (New, Contacted, Follow-Up Pending, Closed)
  - Follow Up Date
